# 公式与阈值库（统一口径）

定位：把容易自由发挥的计算收窄到统一口径。凡涉及下列指标，必须引用本文件；不可计算时写入 `数据缺口`，不得硬算。

## 一、通用计算纪律

- 所有公式结果标注：公式口径、数据来源、数据期间、确定性等级。
- 估算参数必须明示；敏感参数至少做“低/中/高”三档情景。
- 行业不适用时，以 `sector-adapters.md` 为准，禁止套默认公式。
- 模型分数只作预警，不作事实定性；尤其不得由模型直接推出“造假”“爆雷”“低估”。

## 二、基本面公式

### ROIC

```text
NOPAT = EBIT * (1 - 有效税率)
投入资本 = 有息负债 + 所有者权益 - 货币资金及等价物
ROIC = NOPAT / 期初期末平均投入资本
```

规则：
- 金融、保险、券商、类金融平台不使用 ROIC-WACC 作为主指标。
- 若 EBIT、税率、有息负债或现金口径缺失，ROIC 标“不可可靠计算”。
- 只比较同口径、同会计准则、同币种数据。

### WACC

```text
股权成本 Re = 无风险利率 + Beta * 股权风险溢价 + 特定风险调整
债务成本 Rd = 利息支出 / 平均有息负债
WACC = E/(D+E)*Re + D/(D+E)*Rd*(1 - 所得税率)
```

默认参数纪律：
- 无风险利率：使用对应市场 10 年期国债/政府债收益率，并标取值日。
- Beta：优先用可靠二手数据；无可靠 Beta 时不得出精确 WACC，只给情景区间。
- ERP：使用明确来源的全市场 ERP；无法取数时不得做“ROIC-WACC 差值”结论。

## 三、估值公式

### 历史分位

```text
历史分位 = 历史样本中低于当前值的观测数 / 有效观测数
```

规则：
- 至少 3 年月度样本；低于 36 个有效样本时标“样本不足”。
- 发生主营切换、会计口径重大变化、行业周期换挡时，必须提示“历史 band 参考性下降”。

### 反向 DCF

输入：
- 当前市值或 EV
- 最近一年 FCF 或 NOPAT
- 折现率（WACC 或情景折现率）
- 终值增长率
- 显性期年数（默认 5 年或 10 年，需写明）

输出：
- 反推隐含收入/利润/FCF 增速区间，而不是目标价。

禁止：
- FCF 长期为负、财务公司、地产 NAV 模型、18A/18C 未商业化公司，不做反向 DCF。
- 缺少折现率或 FCF 基数时，不做反向 DCF。

## 四、排雷模型

### Beneish M-Score

```text
M = -4.84 + 0.920*DSRI + 0.528*GMI + 0.404*AQI
    + 0.892*SGI + 0.115*DEPI - 0.172*SGAI
    + 4.679*TATA - 0.327*LVGI
```

阈值：
- M > -1.78：进入统计预警区间。
- M <= -1.78：未进入该模型预警区间。

规则：
- 只写“进入/未进入统计预警区间”，不得写“操纵利润”。
- 金融、保险、缺少完整两期报表项目的公司不计算。

### Altman Z-Score

制造业上市公司原版：

```text
Z = 1.2*WC/TA + 1.4*RE/TA + 3.3*EBIT/TA + 0.6*MVE/TL + 1.0*S/TA
```

区间：
- Z > 2.99：安全区
- 1.81 <= Z <= 2.99：灰色区
- Z < 1.81：困境区

非制造业修正版：

```text
Z'' = 6.56*WC/TA + 3.26*RE/TA + 6.72*EBIT/TA + 1.05*BVE/TL
```

区间：
- Z'' > 2.60：安全区
- 1.10 <= Z'' <= 2.60：灰色区
- Z'' < 1.10：困境区

规则：
- 金融、保险、地产慎用；若使用，必须标“参考性弱”。
- 模型区间不能替代公告证据。

## 五、红黄绿扫描

一票否决直接红色提示：
- 非标审计意见（保留、无法表示、否定）
- 被证监/交易所立案调查且未结案
- 重大违规担保未解除
- 港股长期停牌或频繁合股/供股/配售摊薄史

加权项：
- 红项：重大诉讼/资金占用/质押极高/现金流连续恶化/监管处罚
- 黄项：应收存货异常、商誉高企、审计师或 CFO 频繁变更、解禁减持压力

客户版只写“提示项数量与事实”，不要写成买卖依据。

## 六、IPO 风险分档

### A股新股

必要字段：
- 板块同类组近 12 个月首日破发率
- 有效样本数
- 发行 PE vs 行业 PE 偏离
- 超募比例
- 发行规模/流通盘
- 近 1-3 月新股破发率

分档规则：
- 样本数 < 10：不出分档，只列统计不足。
- 同类组破发率 >= 30% 且发行 PE 明显高于行业/可比：统计风险提示偏高。
- 同类组破发率 10%-30% 或定价因子分歧：统计风险提示中性。
- 同类组破发率 < 10% 且定价未明显高估：统计风险提示偏低。

### 港股新股

必要字段：
- 行业 x 发行规模 x 情绪窗口同类组破发率
- 有效样本数
- 基石占比、旧股占比、绿鞋、保荐人历史样本
- 孖展超购截至时点

分档规则：
- 样本数 < 8：不出分档，只列同类样本不足。
- 认购不足、下限定价、高旧股占比、同类组破发率高，任两项命中：统计风险提示偏高。
- 孖展高倍数不得单独降低风险档位；必须结合回拨、融资成本和货源集中度。

固定话术：
> 该档位基于历史样本与公开发行信息，是统计风险提示，不构成申购建议。
