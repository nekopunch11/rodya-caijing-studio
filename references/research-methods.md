# 研究方法论库（A股/港股本土化）

定位：把跨模块复用的研究方法收口到一处。各子模块只写“用哪些方法、怎么落到本模块”，不重复维护方法定义。

## 一、通用硬规则

- **所有使用者共享专业研究判断**：客户合规版保留三情景条件、逻辑支柱方向、证伪条件、利润池和瓶颈判断，但必须同时呈现依据、边界和反面证据，且不得转化为交易动作。
- **每个判断必须能被证伪**：凡写“偏强/偏弱、利润池、瓶颈、风险档位、业绩超预期”，都要有对应的反向触发条件。
- **A股/港股口径分开**：A股关注监管函、质押、限售解禁、行业 PE、中证口径；港股关注流动性、南向资金、美元利率、AH 折溢价、CCASS、供股/合股/配售。
- **同业比较先定同业**：可比公司必须在业务模式、收入结构、市场、周期位置上可比；跨市场比较时标会计准则、币种、流动性折价。

## 二、三情景框架（Bull / Base / Bear）

适用模块：基本面、财报、估值、A股打新、港股打新；产业链可用于题材演化路径。

专业版用法：

| 情景 | 写什么 | 必须包含 |
|---|---|---|
| 乐观 | 哪些关键变量同时向好 | 触发条件、可观测指标 |
| 基准 | 当前公开信息最能支持的路径 | 核心假设、数据依据 |
| 悲观 | 哪些变量恶化会推翻判断 | 证伪条件、风险信号 |

客户合规版：可以保留乐观/基准/悲观的条件与研究含义，不写对应收益或操作；同时列 2–3 个可验证变量，如收入增速、毛利率、订单、政策、同业破发率。

## 三、投资逻辑支柱追踪

适用模块：基本面、财报、估值；组合报告中作为总览页的主线。

输出 3-5 条逻辑支柱，每条固定四列：

| 支柱 | 当前证据 | 最新变化 | 证伪条件 |
|---|---|---|---|
| 例如：毛利率扩张来自产品结构升级 | 毛利率连续 N 期提升，管理层披露高毛利产品占比提高 | 本期继续增强/削弱/中性 | 毛利率回落且产品结构无改善 |

财报模块使用时，必须回答：“本期财报对原有支柱是增强、削弱还是中性？”不要只点评单季数字。

客户合规版：保留支柱增强/削弱/中性的方向含义，但必须紧邻本期证据、反面证据和后续关注指标，不推导交易动作。

## 四、同业四分位对标（Comps Quartile）

适用模块：估值、基本面、产业链、A股打新、港股打新。

步骤：

1. 先定义同业池：3-8 家为宜，必须说明纳入/排除理由。
2. 统一口径：同一期间、同一币种、同一会计口径；港股与 A股跨市场比较时额外标流动性与 AH 折溢价。
3. 对可比指标给出：最大值、75 分位、中位数、25 分位、最小值。
4. 结论只说“处于同业分布的哪个位置”，不把低分位直接翻译为低估。

常用指标：

- 估值：PE、PB、EV/EBITDA、EV/Sales、股息率、FCF yield。
- 经营：收入增速、毛利率、EBITDA margin、ROE、ROIC、FCF margin。
- IPO：发行估值 vs 已上市同业四分位、发行规模 vs 同期新股分布。

## 五、行业关键指标字典

适用模块：基本面、估值、产业链。先问“这个行业真正跑什么指标”，再套财务框架。

| 行业/模式 | 关键指标 | 不宜主看 |
|---|---|---|
| 银行 | 净息差、不良率、拨备覆盖率、资本充足率、存款成本 | 普通 PE、经营现金流 |
| 保险 | NBV、EV/PEV、综合成本率、偿付能力、投资收益率 | 普通 PE、ROIC |
| 券商 | ROE、经纪/投行/自营结构、两融余额、市场成交额 | 普通 PE 静态比较 |
| 地产 | NAV、现金短债比、净负债率、销售回款、土储质量、交付压力 | 普通 PE、反向 DCF |
| 周期资源/化工/航运/养殖 | 商品价格、库存、产能利用率、单位成本、供给纪律 | 低 PE 直接判便宜 |
| 消费 | 同店/渠道、品牌力、毛利率、库存周转、费用投放 | 只看营收增速 |
| 医药成熟公司 | 产品管线、集采影响、销售费用率、现金流、专利到期 | 单纯 PS |
| 创新药/18A | 管线阶段、临床节点、现金 runway、商业化进度 | PE、ROIC |
| 18C/硬科技 | 订单、研发强度、量产良率、客户认证、现金 runway | PE、ROIC-WACC |
| 先进制造/半导体/机器人 | 产能、良率、ASP、客户导入、国产化率、供应链瓶颈 | 单纯题材热度 |
| 平台/互联网 | GMV、take rate、MAU/DAU、履约成本、经营杠杆、监管风险 | 单纯收入规模 |

## 六、价值链利润池分析

适用模块：产业链；基本面可用来判断公司所处环节质量。

利润池不是“价值量高”一个变量，按价值量占比、毛利率/单位利润、集中度、议价权、供给约束五维判断。五项量纲不同，不能直接相乘；各按 1–5 分逐项给证据，输出五维画像、主要优势和关键短板。

判断规则：

- 高价值量 + 高集中度 + 强议价权：利润容易沉淀。
- 高价值量 + 充分竞争：容易增收不增利。
- 供给约束来自产能、良率、认证、牌照、资源、客户导入周期。
- A股题材要防“公告蹭概念”；港股题材要防“流动性折价掩盖基本面分化”。

客户合规版可判断“从价值量、毛利率、集中度和议价权看，利润主要沉淀在 X 环节”，但不可写“最受益/首选”，并需列竞争扩产等反面变量。

## 七、竞争方式分类

适用模块：产业链、基本面、组合 PPT。

先按竞争方式分组，再列公司：

| 分组方式 | 适用场景 | 看什么 |
|---|---|---|
| 平台型/制造型/渠道型/服务型 | 多商业模式并存 | 收入质量、利润率、规模效应 |
| 龙头/二线/新进入者 | 集中度变化 | 份额、价格权、成本曲线 |
| 国产替代/进口替代/全球竞争 | 硬科技、制造 | 客户认证、供应链安全、海外同业 |
| 牌照/资源/成本/品牌/渠道 | A股本土行业 | 政策、资源禀赋、渠道控制 |
| 高流动性龙头/低流动性小票 | 港股 | 流动性折价、南向覆盖、成交额 |

## 八、瓶颈点投资法（特定题材适用）

适用条件：只用于 AI、半导体、机器人、新能源、先进制造、军工电子等“供应链卡位”明显的题材。不用于银行、保险、地产、成熟消费、公用事业等非瓶颈驱动行业。

六步法：

1. **信号去噪**：真实需求是否能落到 BOM、订单、客户认证、长协、产能排期；只有概念标签则降权。
2. **映射财务科目**：需求最先体现在哪一行：收入、ASP、毛利率、在建工程、预收/合同负债、经营现金流。
3. **找错分标的**：市场是否仍按旧行业/旧周期定价，而非按新瓶颈环节定价。
4. **验证错误定价**：是否存在卖方覆盖切换、客户官宣、指数纳入、订单披露等认知修正路径。
5. **建立验证链**：每个季度跟踪订单、ASP、毛利率、产能、竞品扩产、替代技术。
6. **瓶颈强度评分**：确定性、清晰度、纯度、弹性、时间窗，每项 0-5 分。

专业版与客户合规版都可写“瓶颈强/中/弱/无”；客户版必须用供给约束、客户认证、业务纯度、产能和替代技术证据支撑，不写仓位、推荐排序或操作。

## 九、证伪条件与验证链

适用模块：全部。

每个核心判断至少配 1 条证伪条件：

| 判断类型 | 证伪条件示例 |
|---|---|
| 基本面偏强 | ROE 下行且现金流覆盖恶化；护城河证据被竞争削弱 |
| 财报改善 | 改善只来自一次性损益；前瞻指标未跟上 |
| 估值中枢有效 | 增速换挡、利率中枢换挡、商业模式变化 |
| 产业链利润池 | 竞品扩产、价格下行、国产化率停滞、客户认证失败 |
| 打新低风险 | 同类组破发率上升、最终定价上限且高于同业、认购不足 |
| 排雷无重大红旗 | 新监管函、非标审计、资金占用、质押平仓压力出现 |

## 十、排雷恶化因子（Short Screen 本土化）

适用模块：排雷；基本面和财报可做速扫。

跨市场恶化因子：

- 收入下滑或增速连续放缓，且管理层解释不能和行业数据互相验证。
- 毛利率/费用率恶化，利润依赖一次性损益。
- 应收、存货、合同资产增速显著快于收入。
- 经营现金流长期弱于净利润。
- 高估值溢价缺少增长或盈利质量支撑。
- 内部人/重要股东持续减持，或核心管理层频繁变动。

A股额外因子：大股东质押、解禁压力、监管问询/警示函、非标审计、商誉减值、关联交易、资金占用、违规担保。

港股额外因子：供股、合股、折价配售、CCASS 货源集中、核数师频繁更换、长期停牌、低成交额导致的流动性风险。

## 十一、管理层、资本配置与单位经济模型

适用模块：基本面；财报用于增量验证，估值用于正常化假设检查。

### 管理层与治理

- 激励是否与每股价值、ROIC/ROE、现金流或长期经营指标绑定，而非只绑定收入规模。
- 近 3–5 年公开目标兑现记录：承诺、实际、偏差、解释是否可由数据验证。
- 实控人/管理层持股、质押、关联交易、同业竞争、关键人依赖与继任安排。

### 资本配置记分卡

逐项记录再投资、并购、分红、回购注销、增发/可转债与减值结果。评价重点不是“做了什么”，而是资本投入后的增量回报、每股价值变化和稀释成本。无法计算增量 ROIC 时，用新增产能利用率、并购后利润/现金流兑现、回购价格相对估值等替代指标。

### 单位经济模型

按商业模式选择最小经营单元：单客户、单店、单产品、单台设备、单 GW、单 MAU 或单项目。至少呈现收入/毛利、获客与履约成本、回收期、留存/复购及规模扩大后的边际变化。没有披露时明确缺口，不用公司整体毛利率冒充单位经济。

## 十二、预期、调整项与指引可信度桥接

适用模块：财报、估值。

- **预期分布**：除一致预期中位数外，记录覆盖人数、最高/最低或标准差；覆盖过少时不把点估计写成“市场预期”。
- **预测修正**：记录财报前 30/60/90 天收入、利润或关键 KPI 的预测上调/下调方向；财报后区分“当期 beat/miss”和“未来预测修正”。
- **Reported → Adjusted 桥**：逐项列股权激励、重组、减值、公允价值、汇兑、补助、资产处置等调整，给税前/税后影响和是否重复发生。
- **指引可信度**：近 4–8 个可比报告期统计管理层指引区间与实际结果，标兑现、超出、低于及指引区间是否过宽。

## 十三、正常化盈利与估值交叉验证

适用模块：估值、基本面。

- 周期公司用中周期价格、单位成本和产能利用率重建正常化盈利；成长公司区分成熟业务利润与投入期业务亏损；一次性项目从基准盈利中剔除但保留现金影响。
- 使用完全摊薄股本：纳入期权、限制性股票、可转债、配股及已公告融资；同时记录回购注销抵消。
- DCF 必须披露终值占 EV 比例；占比过高时降低结论强度，并扩大折现率/终值增长敏感性。
- 至少两类方法交叉验证：历史分位/同业、反向 DCF、SOTP/NAV、股东回报率或单位经济。结果冲突时解释驱动变量，不做机械平均。
- SOTP 先列分部口径、适配估值锚、净债务/控股折价与不可分摊费用，避免把高估值分部重复计价。

## 十四、风险模型本土校准与误报控制

适用模块：排雷。

- M-Score、Z-Score 等模型先标原始样本、适用行业和中国市场局限；没有本土验证时只作线索排序。
- 每个模型信号都必须回到报表附注、审计意见、监管函或公司行动验证；模型与一手证据冲突时，以一手证据和口径解释为主。
- 区分误报来源：行业营运资本特征、并购导致资产质量变化、会计准则差异、季节性、快速增长期备货、一次性融资。
- 风险记录固定四列：信号、可能解释、验证证据、剩余不确定性。不能排除合理解释时降为“待核实线索”。
- 审计事项按“意见类型—涉及科目/金额—持续期间—整改状态—对现金流/偿债/控制权影响”拆分，不只凭标签一票定性。

## 十五、产业暴露、利润弹性与估值反映

适用模块：产业链；基本面可用于分部分析。

代表公司不只按收入纯度排序，固定拆五层：

1. **业务纯度**：相关收入/资产占比。
2. **增量收入弹性**：行业需求增长 1 单位对公司收入的传导。
3. **增量利润弹性**：价格、产能利用率和经营杠杆对利润的传导。
4. **竞争优势与可持续性**：份额、成本曲线、认证、产能、替代风险。
5. **估值已反映程度**：当前估值相对历史、同业和隐含增长。

利润池五因子是结构化评分框架，不是可直接相乘的精确公式。每项采用 1–5 分并逐项给证据；最终呈现五维画像和关键短板，不输出伪精确乘积。

## 十六、IPO 统计有效性与滚动复盘

适用模块：A股打新、港股打新。

- 每个破发率或表现统计必须带样本选择、窗口、样本数、基准率与置信区间；小样本只作描述，不输出精确概率。
- 分档规则按时间滚动复盘：记录预测时点可得信息、档位、实际暗盘/首日结果和错误类型，禁止用上市后信息回填预测。
- 至少比较“模型/规则档位”与“直接采用板块基础破发率”的增量价值；不能改善基准时，退回描述性统计。
- 港股另记录申请档位—获配结果—融资成本—实际滑点；A股另记录板块、发行估值偏离、发行规模、超募和情绪窗口。
- 复盘用于校准规则，不把个别极端案例直接外推为新阈值。
