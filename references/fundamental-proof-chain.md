# 基本面结论证明协议

> 本文件是 `caijing-fundamental` 判断性结论的唯一证明规则。安全、合规、取数与时效要求仍服从共享层中优先级更高的规则。

## 一、适用范围

十个分析块是最低覆盖，不是封闭框架。所有基本面专业版默认执行本协议，不设置“标准版 / 深研版”双档。

## 二、判断性结论识别

含“强、弱、改善、恶化、领先、落后、可持续、存在护城河、资本回报有效、成长质量高”等定性的命题，或会进入首页研究判断、护城河评分、逻辑支柱和风险定性的表述，必须建立证明链。

纯事实可以不计算，但必须带来源、口径和截至日。无法由证据链支持的形容词不是研究结论。

## 三、结论证明链

统一链条为：

`研究问题 → 原始数据 → 口径调整 → 公式或拆解 → 计算结果 → 历史坐标 → 同业坐标 → 研究判断 → 反面证据 → 证伪条件`

每条链必须记录：编号、关联分析块、原始数据的来源等级与页码、状态（完整 / 断裂）、断点和缺失数据。历史坐标或同业坐标确实不适用时，必须记录不适用原因和替代坐标。

### 最小结构

```yaml
- 编号: "P-01"
  研究问题: ""
  关联分析块: ""
  原始数据: []
  口径调整: ""
  公式或拆解: ""
  计算结果: ""
  历史坐标: ""
  同业坐标: ""
  研究判断: ""
  反面证据: []
  证伪条件: []
  状态: "完整|断裂"
  断点: ""
  缺失数据: []
```

## 四、结论 Gate

只有状态为“完整”的链可以输出研究判断。推导链状态为“断裂”时，停止对受影响事项下定性结论；只保留已核实事实、断点、缺失数据、受影响结论和替代观察指标。

不得用带强弱方向的条件判断维持结论完整感，不得用外部预测值补成公司实际数据，也不得用总公司指标冒充未披露的分部或单位经济。

## 五、框架外扫描

完成标准分析块后必须扫描：

1. 与典型同业不同的资产、负债或收入结构；
2. 能解释大部分利润但未被标准框架捕捉的变量；
3. 使常规指标失真的会计口径、交易结构或监管安排；
4. 足以改变原逻辑的新业务、重大诉讼、关联交易、技术路线或资本动作；
5. 财务数据与商业现实不匹配的异常。

没有重大事项也要记录“已扫描，未发现通过重要性 Gate 的事项”。

## 六、重要性 Gate

框架外事项满足任一条件时，新增正文分析块并建立独立证明链：

- 可能改变收入、利润或现金流的主要来源；
- 可能改变资本结构、控制权或每股价值；
- 可能推翻护城河评分或投资逻辑支柱；
- 常规财务指标因它而失真；
- 公司、审计机构、监管机构或管理层讨论将其列为重大事项。

未通过重要性 Gate 的事项进入附录观察清单，不扩写正文。

## 七、行业适配器覆盖

允许覆盖默认适配器，但必须写明：默认指标的失真原因、采用的替代指标、替代指标如何连接研究判断，以及替代方法的局限。框架外事项不降低证据标准。

## 八、正文与附录（证明链是幕后工序，不是正文体裁）

**铁律**：证明链是保证结论可复核的内部工序，不是读者阅读的文体。报告读者要的是研究结论与洞察；证明链的字段结构一律沉入附录，正文只呈现被它支撑的判断本身。

正文写法——以研报叙事呈现：用自然语言把"因为看到 A、B、C（关键数字，带来源脚注），所以判断 D"讲成连贯论证；关键数字融进句子，判断句末用证明链编号回指（如 `[P-01]`）以便反查。**正文严禁出现证明链结构词**：链状态、嵌入证明链、原始数据与口径、口径调整、计算与比较坐标、拆解方法、关键推导、未完成推导、研究判断（作为标签）、证据锚 等，也不得把一条链的字段逐条罗列成正文小节。一句判断最多附一句"依据/边界"的自然语言收口，不摆字段表。

**断裂链在正文**：只用一句自然语言说明"某判断因缺 X 数据暂不下定性，需持续跟踪 Y"，不在正文摆断裂链结构或"未完成推导"小节；断点与补数清单进附录。

附录写法——集中台账：每条证明链在附录以完整结构呈现（编号、研究问题、原始数据+来源页码、口径调整、公式或拆解、计算结果、历史坐标、同业坐标、研究判断、反面证据、证伪条件、状态、断点、缺失数据），另附完整同业池、重复计算、断裂链和补数清单。正文、附录和首页使用同一证明链编号。

**不写自述方法论**：报告不设"分析框架与证明链地图""证明链状态表""审核规则"等向读者解释自身构造的章节。证明链地图、链状态汇总、首页反查规则属于内部质检，只存在于内核 JSON 和附录，不进正文。

## 九、首页反向审查

首页每一句判断必须能定位到一条或多条状态为“完整”的证明链。无法反查的判断删除或降级为事实；只引用断裂链的判断不得进入首页。

## 十、完成自检

- 判断性结论证明覆盖率为 100%；
- 强判断关联的断裂链为 0；
- 关键计算可由报告中的原始数据、口径和公式复算；
- 历史与同业坐标齐全，或有不适用原因和替代坐标；
- 每条核心逻辑同时有支持证据、反面证据和证伪条件；
- 框架外扫描已记录；
- 行业适配器采用或覆盖理由已披露。
