# 行业适配规则（哪些指标不能硬套）

定位：先判行业，再选择指标。默认工业/消费公司框架不能直接套到金融、地产、未盈利科技、生物医药和强周期行业。

## 一、执行规则

1. 每次个股分析先标注行业类型。
2. 对照本文件决定“主指标、禁用指标、替代指标”。
3. 禁用指标不得出现在关键数字和客户版卡片里。
4. 若公司跨行业，按利润/资产/收入贡献最大的业务定主口径，其余业务分部说明。
5. 需要更细的行业关键指标字典时，读取 `research-methods.md` 第五节；先用行业真正跑的指标，再套财务框架。

## 二、行业适配表

| 行业类型 | 主指标 | 禁用或慎用 | 替代看点 |
|---|---|---|---|
| 银行 | PB-ROE、不良率、拨备覆盖率、净息差、资本充足率 | PE、ROIC-WACC、经营现金流/净利润 | 资产质量、息差、资本约束、存款结构 |
| 保险 | EV/PEV、NBV、偿付能力、投资收益率 | PE、ROIC、普通 DCF | 新业务价值、资产负债久期、利率敏感性 |
| 券商 | PB、ROE、经纪/投行/自营结构 | 普通 PE 分位、经营现金流 | 市场成交、资本中介、杠杆与自营波动 |
| 地产 | NAV、PB、净负债率、现金短债比、预收款 | 普通 PE、反向 DCF | 土储质量、交付压力、融资到期、销售回款 |
| 周期资源/化工/航运/养殖 | PB、EV/EBITDA、单位成本、产能利用率 | 低 PE 直接判断便宜 | 周期位置、库存、价格弹性、供给纪律 |
| 公用事业/高速/电力 | PE、PB、股息率、现金流稳定性 | 高成长 PEG | 监管定价、资本开支、负债期限 |
| 消费/医药成熟公司 | PE、ROE、现金流、毛利率、渠道质量 | 单纯 PS | 品牌、渠道、产品周期、费用投放 |
| 创新药/18A 生物科技 | 管线阶段、现金 runway、临床里程碑 | PE、ROIC、反向 DCF | 适应症空间、临床数据、现金消耗、商业化能力 |
| 18C/硬科技未盈利 | PS、EV/Sales、订单、研发强度、现金 runway | PE、ROIC-WACC | 技术验证、客户集中度、量产能力 |
| SaaS/平台型公司 | ARR/MRR、留存率、毛利率、CAC 回收期 | 传统资产周转率 | 增长质量、单位经济模型、监管风险 |
| 控股平台/多元集团 | SOTP、分部估值、折价率 | 单一 PE/PB 结论 | 分部资产质量、治理折价、现金流上收能力 |

## 三、证明链路由

| 公司类型 | 默认经营桥 | 价值/质量验证 |
|---|---|---|
| 制造业 | 销量 × ASP × 产品结构；材料/制造成本；产能利用率 | 单位毛利、资本开支、增量产能回报 |
| 消费/零售 | 店数/销量 × 客单；同店；渠道和库存 | 店效、费用率、现金转换 |
| 软件/平台 | 用户 × 留存 × ARPU；获客与服务成本 | LTV/CAC、回收期、经营杠杆 |
| 银行 | 生息资产 × 净息差；手续费；信用成本 | 不良生成、拨备、资本充足率、ROE |
| 保险 | NBV、承保利润、投资收益 | 偿付能力、资本生成、股东回报 |
| 研发型公司 | 技术/临床里程碑 × 商业化概率；现金消耗 | runway、融资稀释、里程碑兑现 |

允许覆盖默认适配器，但必须记录：默认指标的失真原因、采用的替代指标、替代指标如何连接研究判断、替代方法的局限。不能因为公司特殊就省略证明链，也不能因为模板存在就强套不适用指标。

## 四、客户版处理

- 客户卡片优先展示该行业最容易理解的 2–3 个指标；客户版 docx/PPT 可保留支持研究判断所需的完整指标。
- 客户版可以简化复杂模型推导，但必须保留模型结论、关键假设、适用边界和可能误报。
- 行业特殊风险必须至少列 1 条，例如银行资产质量、地产交付、创新药现金消耗。
