# 财经·财报

## 作用

`caijing-earnings` 用来回答“这季财报行不行 / 财报前该看什么”。它聚焦财报发布后的增量信息，快速拆解本期收入、利润、现金流、毛利率、超预期情况和前瞻指标；财报未披露时可切换为财报前预览。

## 适合场景

- 年报、中报、季报刚发布，需要快速点评。
- 想判断业绩是高于、符合还是低于可比基准。
- 需要从合同负债、订单、资本开支、存货结构里找下一期线索。
- 财报还没发，需要先整理关键指标、三情景和待验证点。

## 核心优点

- 超预期判定有三层基准：一致预期、公司指引、历史季节性推算。
- 不只看利润增速，还用扣非净利、经营现金流、应收和存货验证利润质量。
- 港股半年报频率与 A 股季报频率分开处理，避免错误环比。
- 财报后回到投资逻辑支柱，判断本期数据是增强、削弱还是中性。
- 财报前预览只列关键指标、三情景变量和证伪条件，不做涨跌预测。
- 没有本期财报原文时标注数据档次与需补材料，不硬做超预期判断。
- 增加一致预期覆盖人数与分布、30/60/90 天预测修正、reported 到 adjusted 调整项桥，以及近 4–8 期管理层指引兑现记录。

## 合规边界

- 只能说“高于/符合/低于所用基准”，不能写“所以会上涨/下跌”。
- 客户合规版保留相对预期、盈利质量和逻辑变化的研究判断，不把财报结果翻译成操作建议或股价预测。
- 管理层表态、股价反应只做事实记录，不引申为交易信号。

## 默认产出

- 专业版：财报点评报告，含速览、预期桥、超预期判定、reported→adjusted 桥、质量验证、分项拆解、预测修正、指引可信度和逻辑支柱影响；财报前可出预览稿。
- 客户版（按需，默认只出专业版，用户要转客户时再出）：财报速览卡结构稿 + 150-300 字转发文案。
