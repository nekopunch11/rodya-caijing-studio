# 财经·基本面

## 作用

`caijing-fundamental` 用来回答“这家公司未来值不值得拥有、市场哪里可能看错”。面向 A 股和港股上市公司，产出一份**论点驱动、前瞻、研究型**的深度研究报告，而不是复述公司资料的结果式体检。

## 适合场景

- 客户或自用，需要对一家公司建立有观点的深度全貌。
- 想知道公司质地之外，“市场共识 vs 我的判断”差在哪、什么会证明我错。
- 先做综合深研，再决定是否接力到估值或排雷模块。

## 核心特点

- **有论点、有分歧**：开篇给方向性判断与整体置信度，并明写市场/卖方共识信什么、本报告哪里不同、为什么（variant view）。
- **胜负手深挖**：识别 2–3 个决定未来 3 年盈利与估值的变量，量化钻透并给估算区间＋置信度＋证伪；数据不全就三角测算，绝不留空。
- **向前看**：三情景（Bull/Base/Bear）带驱动数字，反向 DCF 反推“什么被 price in”，估值矩阵多方法交叉。
- **十四段全面框架**：投资论点与分歧、公司画像、商业模式、护城河（五源评分）、财务体检（杜邦/ROIC–WACC/利润桥）、现金流、成长与单位经济、管理层与资本配置、前瞻情景、估值与隐含预期、多投资哲学对撞、风险与 Pre-Mortem、长期跟踪、附录；每段有真分析，不是一句结论。
- **研究型叙述、结论先行**：每段先给结论，再推过程与判断依据（不用“先提问后回答”）；证明链只作幕后可复核，台账进附录、正文不摆字段。
- **一手证据分级**：Tier1 主源（公告/业绩说明会/纪要）优先，核心数字回溯 Tier2（年报/审计/附注），Tier3（卖方/媒体）仅佐证。
- 已接入行业适配规则；ROIC/WACC/反向 DCF 等统一走公式阈值库。

## 输出纪律（承父层北极星）

- **全面详实靠结构不靠字数**：“覆盖但很薄/结论式/凑字数注水”同为失败。
- **不预设页数、不压字号**；正文 ≥10.5pt。
- **最高不变量**：任何 gate、合规、时效或数据降级只降结论置信度或收对客表达边界，永不把专业输出削成简略或不出。

## 合规边界

- 只做客观数据研究，不给买卖建议；专业版可写“基本面偏强/中性/偏弱”等数据指向，不转化为操作动作。
- 客户合规版保留有证据的方向判断、依据、边界与风险，不出现目标价、操作指令或收益暗示；IRR/Action Price 仅自用、且须过研究安全 Gate。

## 默认产出

- 专业版：十四段前瞻深度研报 docx（论点与分歧 → 胜负手 → 前瞻情景 → 估值矩阵 → 多哲学对撞 → 风险 Pre-Mortem → 背景 → 附录）。
- 客户版（按需）：体检卡结构稿 + 150–300 字转发文案；从研究版降级得到，不反向削弱研究版判断力度。
