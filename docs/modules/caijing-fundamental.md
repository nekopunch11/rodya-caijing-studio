# 财经·基本面

## 作用

`caijing-fundamental` 用来回答“这家公司好不好”。它面向 A 股和港股上市公司，围绕商业模式、护城河、财务质量、成长性、阶段坐标、估值简版和风险速扫，生成一份客观公司体检。

## 适合场景

- 客户问到一家公司，投顾需要快速建立全貌。
- 需要判断公司赚钱能力、利润质量和持续性。
- 想先做综合体检，再决定是否接力到估值或排雷模块。

## 核心优点

- 用“三问主线”组织分析：怎么赚钱、赚的是不是真钱、还能不能持续赚。
- 先定义行业关键指标，再套财务体检，避免所有公司都用同一把尺。
- 把护城河从主观描述拆成五源评分，并要求每一分都有证据。
- 专业版沉淀 3-5 条投资逻辑支柱，并为每条支柱配证伪条件。
- 已接入行业适配规则，银行、保险、地产、周期、未盈利科技等不会硬套通用指标。
- ROIC、WACC 等计算统一走公式阈值库，减少不同 Agent 自由发挥。

## 合规边界

- 只做客观数据体检，不给买卖建议。
- 专业版可以写“基本面偏强/中性/偏弱”等数据指向，不能转化为操作动作。
- 客户版只展示事实数字、来源、风险提示和免责声明，不出现目标价、操作指令或收益暗示。
- 关键数据不足时按取数降级规则处理，不硬出结论。

## 默认产出

- 专业版：基本面体检报告，含一页概要、商业模式、护城河、财务体检、成长性、阶段坐标、投资逻辑支柱、估值与风险简版。
- 客户版：体检卡结构稿 + 150-300 字转发文案。

## 执行事实源

实际执行规则以 `caijing-fundamental/SKILL.md` 为准；公式、行业适配、数据降级和客户版合规降级以 `references/` 中对应文件为准。
