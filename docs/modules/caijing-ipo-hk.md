# 财经·港股打新

## 作用

`caijing-ipo-hk` 回答两个问题：这只港股新股破发统计风险多大；该怎么打、打多少。它基于招股书、发售结构、保荐人、基石、孖展热度和同类样本做体检，并在自用模式下跑计算引擎给出申购方案。

## 双模式（2026-07-09 起）

- **自用模式（默认）**：完整打新顾问——用户档案、券商费率、因子体检、配售证据、风险预算、三档条件性方案、明确推荐与执行清单。具体资金建议必须先满足安全 gate 的前置条件。
- **对客模式**：只出客观体检与统计风险提示档位，不给申购建议，客户版按合规降级。

原「零个人资金逻辑」铁律已经用户批准删除（变更队列留痕）；合规约束保留在对客输出上。

## 适合场景

- 港股新股招股期，需要快速看发行结构并决定打不打、打多少。
- 需要比较基石占比、旧股占比、绿鞋、保荐人历史项目、发行估值同业四分位和同类组破发率。
- 18A/18C 未盈利公司需要单独提示制度和商业化风险。

## 核心优点

- 内置 2026 年制度现状：2025-08 机制 A/B 新规、FINI T+2 时间线；18C 使用其指定公开发售分配机制，不能按普通发行人自由采用机制 B；执行时仍需刷新最新规则。
- 孖展超购看绝对金额不看裸倍数——机制 B 小公开盘下倍数天然虚高，高超购仍破发的案例常见。
- 发行估值放进已上市同业 P75/中位/P25 中看，并同时考虑流动性、AH 折溢价和基石质量。
- 不再用超购倍数和估算人数拼“中签压力公式”；优先读取该股配发公告，其次使用至少 3 只同机制、相近公开盘和入场费的申请档位与配售结果，证据不足只给条件性区间。
- 费用低不等于应该扩大认购；资金上限同时受边际期望、最大损失、融资/抵押安全、流动性与组合资金冲突约束。
- 主观权重只称证据评分；未经训练、样本外验证和概率校准，不输出精确破发概率。压力测试使用常规/压力/极端情景并标样本局限。

## 合规边界（仅对客输出）

- 客户版不写"值不值得打""闭眼打""该打几手"等行为建议。
- 客户版只呈现发行结构、历史样本和统计风险提示档位。
- 自用模式的方案与推荐不进入任何对客交付物。

## 默认产出

- 自用模式：对话内完整分析（体检 → 配售证据 → 风险预算 → 三档条件性方案 → 推荐与执行清单），需要留档可出 docx。
- 对客模式：港股新股体检报告 docx + 打新速览卡结构稿 + 150-300 字转发文案。
