# 财经·港股打新

## 作用

`caijing-ipo-hk` 回答两个问题：这只港股新股破发统计风险多大；明确需要行动建议时该怎么打、打多少。它默认基于招股书、发售结构、保荐人、基石、孖展热度和同类样本做统一专业体检，仅在行动顾问模式下读取个人参数并运行计算引擎。

## 研究层与行动层

- **统一研究模式（默认）**：不追问身份或资金，先完成发行质量与统计风险体检。
- **行动顾问模式（按需）**：用户明确问“打多少/借多少/选哪档”时，补齐用户档案、券商费率、配售证据与风险预算，再给三档条件性方案和执行清单。
- **客户合规口径**：保留发行质量与统计风险判断、依据和边界，不给申购、融资或处置动作。

原「零个人资金逻辑」铁律已经用户批准删除（变更队列留痕）；合规约束保留在对客输出上。

## 适合场景

- 港股新股招股期，需要快速看发行结构并决定打不打、打多少。
- 需要比较基石占比、旧股占比、绿鞋、保荐人历史项目、发行估值同业四分位和同类组破发率。
- 18A/18C 未盈利公司需要单独提示制度和商业化风险。

## 核心优点

- 内置 2026 年制度现状：2025-08 机制 A/B 新规、FINI T+2 时间线；18C 使用其指定公开发售分配机制，不能按普通发行人自由采用机制 B；执行时仍需刷新最新规则。
- 孖展超购看绝对金额不看裸倍数——机制 B 小公开盘下倍数天然虚高，高超购仍破发的案例常见。
- 发行估值放进已上市同业 P75/中位/P25 中看，并同时考虑流动性、AH 折溢价和基石质量。
- 不再用超购倍数和估算人数拼“中签压力公式”；优先读取该股配发公告，其次使用至少 3 只同机制、相近公开盘和入场费的申请档位与配售结果，证据不足只给条件性区间。
- 费用低不等于应该扩大认购；资金上限同时受边际期望、最大损失、融资/抵押安全、流动性与组合资金冲突约束。
- 主观权重只称证据评分；未经训练、样本外验证和概率校准，不输出精确破发概率。压力测试使用常规/压力/极端情景并标样本局限。
- 滚动记录预测时点、申请档位、实际获配、融资成本、暗盘/首日成交和滑点，复盘误差但不使用上市后信息改写原预测。
- 风险预算拆成现金占用、压力/极端损失、抵押品压力跌幅、追保缓冲和流动性折价，不再只靠“保守/激进”标签。

## 合规边界（仅对客输出）

- 客户版不写"值不值得打""闭眼打""该打几手"等行为建议。
- 客户版呈现发行结构、历史样本、统计风险提示档位及其依据和边界。
- 行动顾问模式的方案与建议不进入任何客户交付物。

## 默认产出

- 统一研究模式：默认专业体检 docx。
- 行动顾问模式：对话内完整分析（体检 → 配售证据 → 风险预算 → 三档条件性方案 → 条件性建议与执行清单），需要留档可出 docx。
- 客户合规口径：按用户指定生成港股新股体检 docx、打新速览卡+文案或 PPT。
