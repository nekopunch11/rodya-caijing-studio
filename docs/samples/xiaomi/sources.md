## 估值与风险待核验

- 小米是多元业务集团，估值主方法应使用 SOTP 与交叉验证，不用单一 PE 直接覆盖手机、互联网与汽车业务。
- 当前股价、完全摊薄股本、回购注销与净现金必须使用带 2026-07-10 日期的数据。
- 风险模块重点核验：手机出货与组件成本、汽车补贴和盈利路径、研发投入、回购与稀释、监管/诉讼/审计事项。
- M-Score 与 Z-Score 只有在两期完整报表项目齐全且模型适用性审查通过后才计算；否则只列模型缺口，不强行出分。
