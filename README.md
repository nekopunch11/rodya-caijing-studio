# 📊 财经内容台 · rodya-caijing-studio

> 给券商投顾 / 客户经理的财经内容生产工具包（A股 + 港股）。
> 一次研究，产出两口径内容：**专业版研报**自己看，**客户版卡片/结构稿+文案**转发前二次审核。
> 对客内容只做客观数据体检，不荐股（港股打新模块另含自用计算模式，可给申购方案）。

## 🧭 为什么做这个

满屏都在喊"目标价""闭眼买""上车"的环境里，投顾真正敢转发给客户的，反而是一份**不含任何买卖建议**的客观体检。

这套工具把合规做成了产品灵魂，而不是免责声明里的装饰：

- ✅ 客户版内容零操作指令、零目标价、零收益承诺——只有数据、来源和风险提示
- ✅ 判断权交还客户，工具只负责把事实摆清楚
- ✅ 每份输出自带来源标注、数据截至日和免责声明，**复制即用**

同一个逻辑贯穿所有模块：不替任何人做决策，让专业的呈现代替情绪的煽动。

## 🧩 七个模块，各咬一个问题

### 🩺 财经·基本面 `caijing-fundamental` —— 这公司好不好

三问主线：钱从哪来、是不是真钱、还能赚多久。杜邦三因子分解透视盈利来源，ROIC–WACC 利差检验价值创造能力，Morningstar 五源框架把护城河从玄学变成量化评分，盈利现金含量与可持续增长率双重交叉校验；再以产业生命周期、人均 GDP 国际镜像对标与政策位置记录，为公司标定宏观「阶段坐标」。

### 📈 财经·财报 `caijing-earnings` —— 这季财报行不行

超预期判定采用三层基准体系：分析师一致预期 → 公司业绩指引 → 历史季节性推算，无覆盖小票也能量化判定。扣非口径剥离一次性损益，净利润与经营现金流对照验证含金量，应收/存货增速与收入背离侦测渠道压货信号；合同负债、在手订单与资本开支构成前瞻指标体系，在本期数字里找下季线索。

### ⚖️ 财经·估值 `caijing-valuation` —— 现在贵不贵

行业自适应估值锚：周期股规避低 PE 陷阱改用 PB/EV-EBITDA，金融股 PB-ROE 联动，未盈利成长股 PS/PEG——先选对尺子再量。5–10 年历史分位 band 纵向定位，同业控制变量横向对比，反向 DCF 反推当前股价隐含的增长预期（Mauboussin 预期投资法），并以十年期国债利率与全市场 ERP 分位构成「估值环境坐标」——分位便宜不等于便宜，四重价值陷阱检查兜底。

### 🚨 财经·排雷 `caijing-risk` —— 有没有雷

学术模型打底：Beneish M-Score 八因子盈余操纵概率模型 + Altman Z-Score 财务困境预警。叠加 A 股本土化红旗库（大存大贷、非标审计意见、其他应收款资金占用、在建工程久不转固、审计师异动）与港股专属图谱（老千股合股供股史、CCASS 货源集中度）。专业版保留红黄绿总评与一票否决，客户版降级为公开信息提示项——每面红旗必须有公告级证据。

### 🔗 财经·产业链 `caijing-industry` —— 题材链条怎么分、代表公司有哪些

利润池分析（profit pool）定位价值最终沉淀的环节，CR3/CR5 竞争格局与价值量占比双维扫描；渗透率 S 曲线标定题材生命周期（0→1 主题期 / 1→N 兑现期）；代表公司标注「纯度指标」——相关业务收入占比，用客观数据区分真实业务相关和低相关蹭题材。

### 🆕 财经·A股打新 `caijing-ipo-a` —— 新股破发统计风险多大

注册制时代的破发因子模型：发行 PE 行业偏离度、超募比例、流通盘结构、板块分组滚动破发率基准（主板/创业板/科创板/北交所分别统计）、网下报价入围率与弃购率信号，输出统计风险提示档位——给的是历史样本参考，不是拍脑袋预测，更不是申购建议。

### 🎯 财经·港股打新 `caijing-ipo-hk` —— 怎么打、打多少 + 破发统计风险

双模式。体检层：保荐人历史胜率追踪、基石投资者结构分析、老股东套现比例侦测、绿鞋稳价机制核查、孖展超购热度解读（看绝对孖展金额不看裸倍数——机制 B 小公开盘下倍数天然虚高，超购千倍照样破发）、18A/18C 制度风险标签，锚定同类组历史破发率给统计提示档位。计算层（自用模式）：内置 2025-08 机制 A/B 新规与 FINI T+2 时间线、中签压力公式、五大边际条件、券商费用模型（免息/计息分别建模）、三档方案与明确推荐。对客输出仍只出客观体检。

> 模块之间有接力关系：基本面里估值和风险只做简版，想深挖时估值、排雷模块接着上，不重复劳动。

## 📦 每个模块出什么

| 模块 | 专业版（自己看） | 客户版（转发客户） |
|---|---|---|
| 🩺 基本面 | 研报级体检报告 docx | 体检雷达卡 + 文案 |
| 📈 财报 | 财报点评 docx | 财报速览卡 + 文案 |
| ⚖️ 估值 | 估值分析 docx | 分位 band 卡 + 文案 |
| 🚨 排雷 | 排雷报告 docx | 风险提示卡 + 文案 |
| 🔗 产业链 | 产业链梳理 docx | 图谱卡 + 文案 |
| 🆕 A股打新 | 新股体检 docx | 打新速览卡 + 文案 |
| 🎯 港股打新 | 新股体检 docx + 自用申购方案（三档+推荐） | 打新速览卡 + 文案 |

> 多模块组合时产出**汇报 PPT**（晨会、客户沙龙主场）；单模块不出 PPT。视觉模板完成前，默认先交付结构化内核、Markdown 专业稿、客户版文案和卡片/PPT 结构稿。所有形态同源一份「结构化分析内核」——改一处，处处同步。

## 🚀 安装

本包同时兼容 Claude 和 Codex——两家用的是同一套 [agent skills 开放标准](https://agentskills.io)，一份文件两边通用。

### 方式一：npx 一行命令（推荐，全平台通用）

```bash
npx skills add nekopunch11/rodya-caijing-studio
```

[skills CLI](https://github.com/vercel-labs/skills) 是 agent skills 生态的通用安装器，自动检测机器上的 Claude Code / Codex / Cursor 等 agent 并装进对应目录。加 `-g` 装全局；日后 `npx skills update` 一键更新。

### 方式二：让 AI 自己装（最省心）

不想碰命令行？把仓库地址直接发给你的 Claude 或 Codex，加一句话：

```
帮我把这个 skill 装到你的 skills 目录：
https://github.com/nekopunch11/rodya-caijing-studio
```

它会自己 clone 到正确位置。装完可以让它报一遍七个模块的名字，对得上就是装好了。

### 方式三：claude.ai / Claude 桌面版（Cowork）

把文件夹压缩为 zip，在 **设置 → 功能（Capabilities）→ 技能** 中上传。

> 💡 Codex 用户注意：打开本仓库时，根目录 `AGENTS.md` 会自动加载合规硬约束；优先调用父 skill `rodya-caijing-studio`，由父 skill 路由到七个子模块。不同运行器对子目录 skill 的递归发现可能不同。

## 💬 用法

装好后直接说人话，父 skill 自动路由到对应模块：

- 🩺 **基本面**："全面分析一下宁德时代"——客户问到一只票，几分钟拿到自己看的研报和转给客户的卡片。
- 📈 **财报**："茅台出年报了，点评一下"——财报季批量快评，每家一张速览卡，客户群里一天可以专业地出现很多次。
- ⚖️ **估值**："招商银行现在贵不贵"——"能不能买"这种问题也可以丢给它，它只答贵不贵，判断权留给客户。
- 🚨 **排雷**："帮我排查下这家公司有没有雷"——客户拿着小道消息来问时，用红旗清单和公告证据代替拍胸脯。
- 🔗 **产业链**："梳理一下固态电池产业链"——题材热起来时给客户一张图谱，哪些环节和公司真实相关，一目了然。
- 🆕 **A股打新**："XX 明天科创板申购，分析下"——申购日前出体检，给历史样本下的统计风险提示档位。
- 🎯 **港股打新**："这只港股新股怎么打"（贴招股资料）——保荐人、基石、孖展热度、同类组破发率一次扫完；自用模式再算中签压力和三档方案，给明确推荐。
- 📽️ **组合出 PPT**："基本面 + 估值 + 排雷，出个汇报 PPT"——晨会和客户沙龙的主场。
