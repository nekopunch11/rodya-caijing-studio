# 财经内容台 · rodya-caijing-studio

> 给券商投顾 / 客户经理的财经内容生产工具包（A股 + 港股）。
> 一次研究，产出两口径内容：**专业版研报**自己看，**客户版卡片+文案**直接转发。
> 只做客观数据体检，不荐股。

## 为什么做这个

满屏都在喊"目标价""闭眼买""上车"的环境里，投顾真正敢转发给客户的，反而是一份**不含任何买卖建议**的客观体检。

这套工具把合规做成了产品灵魂，而不是免责声明里的装饰：

- 客户版内容零操作指令、零目标价、零收益承诺——只有数据、来源和风险提示
- 判断权交还客户，工具只负责把事实摆清楚
- 每份输出自带来源标注、数据截至日和免责声明，**复制即用**

同一个逻辑贯穿所有模块：不替任何人做决策，让专业的呈现代替情绪的煽动。

## 一次研究，三种产出

每次分析只产出一份「结构化分析内核」，docx / 卡片 / PPT 都是这份内核的不同渲染——改内核一处，三种产出同步更新，永远不会互相打架。

| 形态 | 给谁 | 何时产出 |
|---|---|---|
| 研报级 docx | 投顾自己看（可带倾向判断） | 单模块默认 |
| 卡片图 + 文案 | 转发客户（纯客观口径） | 单模块默认 |
| 汇报 PPT | 晨会、客户沙龙、内部汇报 | 多模块组合时 |

## 七个模块，各咬一个问题

| 模块 | 只回答一个问题 | 方法论内核 |
|---|---|---|
| 财经·基本面 `caijing-fundamental` | 这公司好不好 | 杜邦拆解 · ROIC vs WACC · 护城河五源评分 |
| 财经·财报 `caijing-earnings` | 这季财报行不行 | 超预期三层基准 · 扣非与现金流验证 · 前瞻指标 |
| 财经·估值 `caijing-valuation` | 现在贵不贵 | 行业估值锚适配 · 历史分位 · 反向 DCF 隐含预期 |
| 财经·排雷 `caijing-risk` | 有没有雷 | M-Score/Z-Score · 本土红旗库 · 一票否决项 |
| 财经·产业链 `caijing-industry` | 受益标的有哪些 | 利润池分析 · 纯度指标 · 渗透率 S 曲线 |
| 财经·A股打新 `caijing-ipo-a` | 新股破发风险多大 | 发行 PE 对比 · 板块分组破发率 · 弃购率信号 |
| 财经·港股打新 `caijing-ipo-hk` | 新股值不值得打 | 保荐人胜率 · 基石占比 · 绿鞋 · 18A/18C 标签 |

模块之间有接力关系：基本面里估值和风险只做简版，想深挖时估值、排雷模块接着上，不重复劳动。

## 安装

本包同时兼容 Claude 和 Codex——两家用的是同一套 [agent skills 开放标准](https://agentskills.io)，一份文件两边通用。

### 方式一：git clone（推荐，更新方便）

**Claude Code**（个人全局，所有项目可用）：

```bash
git clone https://github.com/<你的用户名>/rodya-caijing-studio.git ~/.claude/skills/rodya-caijing-studio
```

**Codex**（CLI / IDE / App，个人全局）：

```bash
git clone https://github.com/<你的用户名>/rodya-caijing-studio.git ~/.agents/skills/rodya-caijing-studio
```

**项目级**（随仓库分发给团队）：把整个文件夹放进项目的 `.claude/skills/`（Claude）或 `.agents/skills/`（Codex）。

Windows 下 `~` 即 `C:\Users\<用户名>`。更新时进目录 `git pull` 即可。

### 方式二：下载 ZIP

GitHub 页面点 **Code → Download ZIP**，解压后把整个 `rodya-caijing-studio` 文件夹放进上述对应目录。

### 方式三：claude.ai / Claude 桌面版（Cowork）

把文件夹压缩为 zip，在 **设置 → 功能（Capabilities）→ 技能** 中上传。

> Codex 用户注意：打开本仓库时，根目录 `AGENTS.md` 会自动加载合规硬约束；`$` 提及可显式调用各模块，按问题描述也会自动匹配。

## 用法

装好后直接说人话：

- "全面分析一下宁德时代" → 基本面
- "茅台出年报了，点评一下" → 财报
- "招商银行现在贵不贵" → 估值
- "帮我排查下这家公司有没有雷" → 排雷
- "梳理一下固态电池产业链" → 产业链
- "XX 明天科创板申购，分析下" → A股打新
- "这只港股新股怎么样"（贴招股资料） → 港股打新

## 推荐用法

**日常个股响应**：客户问到哪只，单模块跑一遍，几分钟拿到"自己看的"和"转给客户的"两个版本。

**财报季**：财报模块批量快评，每家出一张速览卡，客户群里一天可以专业地出现很多次。

**晨会 / 客户沙龙**：基本面 + 估值 + 排雷组合，直接出汇报 PPT——单模块不出 PPT，杀鸡不用牛刀。

**打新服务**：A股/港股新股各有速览卡，招股期内数据带截至时点，破发风险给的是历史统计参考，不是拍脑袋。
