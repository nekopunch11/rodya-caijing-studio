# 财经·基本面（caijing-fundamental）

**只咬一个问题：这公司好不好。**七模块中最重的一个（全面体检）。

**边界（接力关系）**：估值只做简版（贵不贵深挖 → `caijing-valuation`）；风险只做红旗速扫（排雷深挖 → `caijing-risk`）。不越界重复劳动。

执行前先读共享层：`../references/research-safety-gate.md`、`../references/compliance.md`、`../references/compliance-rendering.md`、`../references/output-spec.md`、`../references/data-sourcing.md`、`../references/data-fallback.md`、`../references/sector-adapters.md`、`../references/research-methods.md`。涉及 ROIC/WACC 时读 `../references/formulas-and-thresholds.md`。

**执行硬约束**：本模块是重型基本面体检，不是聊天短评。用户未明确要求“只在聊天里回答”时，必须生成专业版 `.docx`。近 3 年年报 + 最新一期财报不足时，仍按十个分析块输出专业版文件，但受影响结论必须按 `data-fallback.md` 标为“待核实判断/条件性判断”并列出补充材料；不得用 3–5 段结论冒充研报。

## 一、三问主线（分析的叙事脊柱，docx 和文案都沿这条线讲）

1. **赚不赚钱、怎么赚的**——盈利水平 + 利润来源结构（杜邦、ROIC）
2. **赚的是不是真钱**——利润的现金含量（盈利质量）
3. **能不能持续赚**——护城河 + 行业空间 + 成长引擎

## 二、分析框架（十个分析块，即内核的 `分析块` 清单）

### 1. 公司速览

主营业务一句话、市值、市场/币种、股东结构（实控人、前十大、机构占比）、上市年限；先按 sector-adapters.md 与 research-methods.md 判断行业类型和**行业真正跑的 3–5 个关键指标**，后文只围绕这些指标展开。

### 2. 商业模式

怎么赚钱（产品/服务 × 收费方式）、客户是谁（2C/2B/2G，集中度）、上下游议价地位（预收/应付 vs 应收/预付的占款结构）。

### 3. 护城河五源评分（Morningstar 框架，把玄学变 checklist）

| 来源 | 判断依据示例 |
|---|---|
| 无形资产 | 品牌溢价、专利、特许牌照 |
| 转换成本 | 客户迁移代价、生态绑定 |
| 网络效应 | 用户越多价值越大的结构 |
| 成本优势 | 规模/工艺/资源带来的毛利率持续领先 |
| 规模效应（有效规模） | 市场只容得下少数玩家 |

每源 0–2 分（0 无 / 1 存疑或减弱中 / 2 明确且有数据佐证），总分 0–10，**每一分都要给证据**，无证据不给分。

### 4. 财务体检（三层）

- **盈利能力——杜邦拆解**：ROE = 净利率 × 总资产周转率 × 权益乘数，近 5 年三因子趋势，回答"利润是靠产品力、效率还是杠杆赚的"。
- **价值创造——ROIC vs WACC**：先按 sector-adapters.md 判断是否适用；适用时按 formulas-and-thresholds.md 计算并标假设。金融、保险、券商、类金融平台不把 ROIC-WACC 放入关键数字。
- **盈利质量**：经营现金流 / 净利润（近 3 年均值，<0.8 打问号）；收现比（销售收现 / 营收）；扣非净利润占比（利润是否靠一次性损益撑着）。
- **财务结构**：资产负债率（行业口径）、有息负债与货币资金对比、流动比率。

### 5. 成长性

收入/归母净利 5 年 CAGR；增长来源拆解（量 / 价 / 新业务 / 并表）；再投资视角 sanity check：可持续增长 ≈ ROE × 留存率，宣称的高增长若远超此值，必须能指出额外资金来源（融资/杠杆），指不出则打问号。

### 6. 估值简版（一段话，不展开）

当前 PE/PB + 所处历史分位一句话。深挖交给 `caijing-valuation`。

### 7. 风险速扫（红旗 3–5 项，不展开）

速查：商誉/净资产比、大股东质押比例、经营现金流连续为负、近期减持/解禁、非标审计意见。命中即列入内核 `风险`。深挖交给 `caijing-risk`。

### 8. 阶段坐标（公司是国家阶段的函数，服务三问之三）

- **产业生命周期**：行业渗透率与增速所处阶段（导入/成长/成熟/衰退），口径与产业链模块一致。
- **国际镜像对标**：按人均 GDP 阶段对标先行国家（日本/韩国/美国）同阶段该行业的渗透率与形态，作为空间参考（历史数据客观呈现，标 B/C 级）。
- **政策位置记录**：当前政策取向（支持/中性/规范），只记录文件与公开表态事实，**不预测转向**。

**铁律**：本块只做坐标呈现（"当前处于什么位置"），禁止任何"因此将会…"的预测句式。

### 9. 管理层、资本配置与单位经济模型

按 research-methods.md 第十一、十三节执行：检查管理层激励与公开目标兑现记录；把再投资、并购、分红、回购、融资稀释做成资本配置记分卡；选择单客户/单店/单产品/单产能等最小经营单元验证单位经济；对周期、并购或一次性项目影响较大的公司给出正常化盈利桥。无披露时列数据缺口和替代指标，不用整体毛利率代替单位经济。

### 10. 投资逻辑支柱与证伪链（专业版主场）

按 research-methods.md 输出 3–5 条逻辑支柱，每条必须含：

| 字段 | 要求 |
|---|---|
| 支柱 | 一句话讲清核心逻辑，如“盈利质量来自高现金转换” |
| 当前证据 | 至少 1 个 A/B 级数据或公告证据 |
| 最新变化 | 增强 / 削弱 / 中性（仅专业版） |
| 证伪条件 | 哪个公开指标变化会推翻该支柱 |

客户版不展示“增强/削弱”判断，只把证伪条件降级为“后续可关注指标”。

## 三、取数清单（对照 data-sourcing.md 执行）

- 近 5 年年报关键数据：营收、归母净利、扣非净利、经营现金流、ROE 及杜邦三因子、毛利率、资产负债表要项
- 最新一期财报（A股最近季报 / 港股最近中期或年报）
- 当前行情与估值（PE/PB 及历史分位）
- 股东结构、质押、近 12 个月增减持
- 行业空间与格局（用于护城河与成长判断）
- 行业渗透率、先行国家同阶段对标数据、相关政策文件（阶段坐标用）
- 行业关键指标：按 research-methods.md 第五节选取（如银行 NIM/不良率，硬科技订单/良率/客户认证，消费渠道/库存/毛利率）
- 管理层激励与公开目标、并购/资本开支/分红回购/融资稀释记录；可得时取最小经营单元的收入、毛利、获客/履约成本、留存和回收期

## 四、内核字段（继承通用 schema 的具体化）

- `关键数字` 固定 7 条：ROE（及 5 年趋势）、ROIC vs WACC 差值（行业不适用时替换为 sector-adapters.md 指定主指标）、经营现金流/净利润、营收 5 年 CAGR、毛利率、护城河总分、资产负债率。
- `分析块` = 上述十个分析块。
- `研究方法.逻辑支柱` 固定 3–5 条；`研究方法.证伪条件` 每条支柱至少 1 条。
- `结论.客观概括` 句式参考："{公司} 近五年 ROE 均值 X%、经营现金流/净利润 Y，护城河评分 Z/10，主要看点与风险各 N 项。"
- `结论.专业判断`（仅专业版）句式参考："基本面偏强/中性/偏弱；多空关键变量：……"

## 五、输出

### 专业版 docx（研报级完整版本）

1. 一页概要（客观概括 + 专业判断 + 关键数字表 + 多空关键变量）
2. 公司与商业模式
3. 护城河五源评分（逐源证据）
4. 财务体检（杜邦 / ROIC vs WACC / 盈利质量 / 财务结构）
5. 成长性
6. 阶段坐标（生命周期 / 国际对标 / 政策位置）
7. 管理层、资本配置、单位经济与正常化盈利
8. 投资逻辑支柱与证伪链
9. 估值与风险简版（注明可接力深挖）
10. 数据附录（来源清单、确定性分级、数据缺口）+ 免责声明

交付要求：

- 正文必须覆盖上述 10 个章节；缺任何章节都要在“数据缺口/未完成项”中明示。
- 篇幅服从信息密度与证据充分度；按 output-spec.md 的分析块覆盖、证据覆盖、来源占比、公式透明、证伪闭环和数据缺口六项完成标准验收，不为凑字数重复扩写。
- 财务体检至少包含近 5 年收入、归母净利、经营现金流、毛利率、资产负债表要项；不能取得则写明数据档次、缺口和替代观察指标，不硬算。
- 护城河五源评分必须逐项给证据；无证据不得给分。
- 估值简版拿不到当前 PE/PB 或历史分位时，只写“估值数据缺口”，不得给贵/不贵判断。
- 最终聊天回复只给 5–8 行摘要 + `.docx` 文件路径 + 数据截至日 + 确定性分级 + 数据缺口；不要把专业稿全文贴成聊天回答。

### 客户版（卡片 + 文案）

- **体检卡**：按 card-components.md 布局——主视觉为五维雷达（护城河/盈利/质量/成长/风险，各维映射规则建卡时固化），关键数字区取 ROE、现金流/净利润、营收 CAGR 三条，风险区取最高严重度 1 条，footer 固定件。
- **文案**（150–300 字）模板：

> 【{公司} 基本面速览】数据显示，{公司} 近五年平均 ROE 为 X%，经营现金流对净利润的覆盖为 Y 倍，营收五年复合增速 Z%。{一句业务/格局的客观事实}。需关注的风险点：{风险 1}。以上为客观数据整理，供参考。
> （固定免责声明）

- 渲染后逐句过 compliance.md 客户版 checklist。

## 六、工作流

1. 按取数清单收集数据（联网 + 用户上传），逐条标来源与确定性；不满足 data-fallback.md 最低材料时继续执行十个分析块，但把受影响结论标为待核实或条件性判断，并写出补充材料清单。
2. 先判行业并应用 sector-adapters.md 与 research-methods.md 的行业关键指标字典，再依十个分析块产出结构化内核（一份，唯一事实源）。
3. 先写完整专业版研报正文，再渲染专业版 `.docx`；若 `.docx` 生成失败，修复或明确改交付原因，不得静默改为短评。
4. **（按需）**用户要转客户或明确要客户版时，才渲染客户版卡片结构稿 + 文案（口径切换规则见 output-spec.md 与 compliance-rendering.md）；否则跳过，只交付专业版并提醒使用者可另出客户版。
5. 合规检查（客户版逐句）。
6. 交付，标 `.docx` 文件路径、数据截至日、确定性分级、数据缺口和未完成项。
